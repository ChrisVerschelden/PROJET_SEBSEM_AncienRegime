# Instruction

## Prerequesite

you should have a valid install of npm, check with :
```
npm -v
```

## Run app in dev mode

go to the WEBSEM_AncienRegime_VISU and in the terminal type 
```
npm i
```
```
npm run dev
```

## Build app

go to the WEBSEM_AncienRegime_VISU and in the terminal type 
```
npm i
```
```
npm run build
```
